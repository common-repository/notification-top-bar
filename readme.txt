== Description ==
**Description:**    

Responsive notification top & alert bar. Use for: Newsletter, social media, notification, call to action, video, countdown. 50+ customizable options.


**TRY IT: See it live [here](http://plugin-boutique.com/notification-top-bar/)**


**Features:**
<ul>
<li>20+ Animations ($5 version feature)</li>
<li>Custom Design ($5 version feature)</li>
<li>Custom Background Color ($5 version feature)</li>
<li>Custom text & link color ($5 version feature)</li>
<li>Custom font styling ($5 version feature)</li>
<li>Custom padding ($5 version feature)</li>
<li>Drop shadow ($5 version feature)</li>
<li>Hide on mobile ($5 version feature)</li>
<li>Only show on mobile ($5 version feature)</li>
<li>Only show X amount of times ($5 version feature)</li>
<li>Hide on specific pages/posts ($5 version feature)</li>
<li>Only show on specific pages/posts ($5 version feature)</li>
<li>Newsletter</li>
<li>Mailchimp, Get Response & Campaign monitor integration.</li>
<li>Feedburner, Aweber, Mad Mimi integration</li>
<li>Social media icons</li>
<li>Countdown</li>
<li>Youtube & Vimeo integration</li>
<li>Shortcode integration</li>
<li>Show/Hide button</li>
<li>Positioning</li>
<li>Track Clicks</li>
<li>Track Views</li>
<li>Visual Editor</li>
</ul>


**Activation:** 
<ul>
<li>Upload the zip and install the plugin.</li>
<li>Click Activate.</li>
<li>Go to Wordpress Admin Panel</li>
<li>Go to "Notification Bar" </li>
<li>Go to "Add New"</li>
<li>Create an awesome notification bar</li>
</ul>
